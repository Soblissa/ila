## Teil 1 — Dein KI-Agent

Du hast jetzt einen KI-Agenten. Du erreichst ihn über Telegram: **@FelgesBot**

**So startest du:**
1. Suche in Telegram nach **@FelgesBot**
2. Schreib ihm eine erste Nachricht
3. Er antwortet mit einem Pairing-Code — tippe den Code zurück in denselben Chat
4. Fertig — ihr seid verbunden

**Was dein Agent darf:**
- ILA-Repo lesen und schreiben
- Figma lesen und analysieren
- Cheko Aufträge erteilen (z.B. "erstelle Screen X für Anforderung Y")
- Andere ILA-Themen bearbeiten
- Private Dinge — bitte aber sparsam damit umgehen, da jede Anfrage Kosten verursacht

**Was er nicht darf:**
- In Figma schreiben oder Screens erstellen
- Entscheidungen ohne Sarah treffen

**Wichtig:** Deine Hauptaufgabe ist es, fachliche Anforderungen für neue Screens zu formulieren — dein Agent hilft dir dabei. Alles was dein Agent mit Cheko bespricht wird dokumentiert.

## Teil 2 — GitHub-Konto erstellen und dem ILA-Repo beitreten

**Schritt 1 — Konto erstellen:**
1. Geh auf **github.com**
2. Klick auf **"Sign up"**
3. E-Mail, Passwort und Benutzernamen wählen
4. Konto bestätigen (E-Mail)

**Schritt 2 — Benutzernamen an Sarah schicken**
Sobald du registriert bist, schick Sarah deinen GitHub-Benutzernamen. Sie lädt dich dann ein.

**Schritt 3 — Einladung annehmen**
Du bekommst eine E-Mail von GitHub mit einer Einladung zum ILA-Repo. Auf den Link klicken und annehmen — fertig.

Danach kannst du das Repo unter **github.com/Soblissa/ila** sehen und dein Agent kann in deinem Namen schreiben.

Mögliche Sonderzeichen in der LUSD
Ergänzend zu den Zeichen auf der PC-Tastatur können Sie in der LUSD die in der Tabelle
aufgeführten Sonderzeichen (ab Kapitel 2.3) speichern.
Bei dem Versuch, andere als in der Tabelle aufgeführte Sonderzeichen zu speichern,
erhalten Sie eine Fehlermeldung.
Tipp
Die Angabe der Codepoints dient der eindeutigen Identifizierung des jeweiligen
UNICODE-Zeichens.
Bei bekanntem Codepoint können Sie das gewünschte Zeichen in diesem Dokument
über die Suchfunktion (Strg + F) finden

Eingabe der Sonderzeichen
So geben Sie die Sonderzeichen in der LUSD ein:
Kopieren und Einfügen
1. Markieren Sie in der Tabelle das gewünschte Sonderzeichen.
2. Drücken Sie die Tastenkombination Strg + C (= kopieren).
3. Gehen Sie an die gewünschte Stelle in der LUSD.
4. Drücken Sie die Tastenkombination Strg + V (= einfügen). Das Sonderzeichen wird
eingetragen.

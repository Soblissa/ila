1.1.5 Liste der Lehrkräfte
In diesem Bericht werden Informationen zu den Lehrkräften Ihrer Schule ausgegeben.

Abb. 8: Beispiel: Bericht LISTE DER LEHRKRÄFTE mit Funktionen

Ausgabeformate

PDF, CSV

Enthaltene Daten

Kürzel Lehrkraft, Vor- und Nachname Lehrkraft, Adresse, Telefon,
Mobiltelefon, Fax, Private E-Mail, Dienstliche E-Mail,,
Geburtsdatum, Dienstbezeichnung, Amtliche Funktion, Fächer in
UV, Teilnahme Schulkonferenz, Personalnummer SAP

Parameter

UV
Version: lang, kurz, mit Funktionen, Raster, zweispaltig
Überschrift: Freitext

Zusätzliche Informationen

Version kurz: In dieser Version werden Kürzel, Name und mögliche
Telefon- bzw. Mobiltelefonnummern der Lehrkräfte ausgegeben.
Version lang: In dieser Version werden Kürzel, Name,
Kontaktdaten, Geburtsdatum, Dienstbezeichnung und die
Personalnummer der Lehrkräfte ausgegeben.
Version mit Funktionen: In dieser Version werden Kürzel, Name,
Kontaktdaten, Geburtsdatum, Dienstbezeichnung, Funktion,
Unterrichtsfächer, Teilnahme an der Schulkonferenz und die
Personalnummer der Lehrkräfte ausgegeben.
Version Raster: In dieser Version wird eine Liste mit den Namen
der Lehrkräfte und einer freien Spalte ausgegeben. Diese Version
kann zum Beispiel als Unterschriftenliste verwendet werden.
Version zweispaltig: In dieser Version wird eine zweispaltige Liste
mit den Namen der Lehrkräfte ausgegeben.
